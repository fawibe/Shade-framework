# Contributing

Thanks for your interest!

## What you can contribute

- Improvements to wording (clarity, neutrality)
- Accessibility and UX improvements
- Bug fixes
- New export formats (still local-only)
- Translations (FR/EN/...)

## How to propose changes

1. Open an Issue describing the change
2. Submit a Pull Request referencing the Issue

## Principles

- Privacy-first: no remote calls, no tracking, no analytics
- Simple static stack: HTML/CSS/JS, Bootstrap allowed via CDN
- Neutral tone (non-partisan, no FOMO)
